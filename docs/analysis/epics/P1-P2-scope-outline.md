## P1 / P2 kapsam taslağı (başlıklar)

Bu doküman sadece kapsam başlıklarını listeler. Detaylar daha sonra `docs/analysis/features/` altında açılır.

### P1 (günlük kullanımın tamamına yakın)

- Saved filters / gelişmiş filtreleme
- Attachments: görüntüleme + yükleme (kamera/dosya)
- Work package ilişkileri (parent/child/related)
- Arama (global + proje bazlı)
- Bildirimlerde gelişmiş deneyim (gruplama/filtreleme) (destekleniyorsa)
- Time tracking: zaman kayıtlarını düzenleme/silme (yetki ve API destekliyse)

### P2 (ileri / nadir / admin)

- Admin ve proje ayarları ekranları
- İleri raporlar / dashboard benzeri ekranlar
- Özel alan yönetimi (custom fields) (varsa)
