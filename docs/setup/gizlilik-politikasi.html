ProjectFlow for OpenProject – Gizlilik Politikası

Son güncelleme: Ocak 2025

ProjectFlow for OpenProject (“uygulama”), OpenProject sunucunuza bağlanan bir mobil istemcidir. Uygulama ücretsiz indirilir; Pro sürümü tek seferlik uygulama içi satın alma ile açılır. Bu politika, uygulamanın verilerinizi nasıl işlediğini açıklar.

1. Toplanan veriler

Uygulama yalnızca sizin girdiğiniz ve cihazınızda saklanan aşağıdaki verileri kullanır:

  • OpenProject instance adresi (URL): Bağlanmak istediğiniz OpenProject sunucusunun adresi.
  • API anahtarı: OpenProject hesabınızdan aldığınız API anahtarı; sunucuyla kimlik doğrulama için kullanılır.
  • Tercihler: Aktif proje, tema (açık/koyu), bildirim ayarları gibi uygulama tercihleriniz.
  • Pro satın alma durumu: Pro’yu satın aldıysanız, bu bilgi yalnızca cihazınızda (ve Google Play hesabınızla eşleşecek şekilde) tutulur; satın alma öncesi/sonrası ek kişisel veri toplanmaz.

Bu veriler yalnızca cihazınızda güvenli depolama (Android keystore tabanlı güvenli depolama) ve uygulama tercih dosyalarında tutulur.

2. Verilerin kullanımı

  • Bağlantı: Instance URL ve API anahtarınız, yalnızca sizin seçtiğiniz OpenProject sunucusuyla iletişim kurmak için kullanılır (ör. iş paketleri, bildirimler, zaman kayıtları).
  • Üçüncü taraflarla paylaşım: Bu veriler reklamcılar, analiz şirketleri veya başka üçüncü taraflarla paylaşılmaz veya satılmaz.
  • Sunucu tarafı: OpenProject sunucunuz kendi politikasına göre verileri işler; uygulama yalnızca sizin adınıza bu sunucuya istek gönderir.
  • Pro ödemesi: Pro sürümü, Google Play üzerinden tek seferlik uygulama içi satın alma ile açılır. Ödeme (kredi kartı vb.) Google tarafından işlenir; uygulama ödeme bilgisi saklamaz veya işlemez.

3. Veri saklama yeri

Tüm yukarıdaki veriler cihazınızda kalır. Proje verileri (iş paketleri, bildirimler vb.) istek üzerine OpenProject sunucunuzdan alınır ve uygulama içinde gösterilir; kalıcı olarak cihazda saklanmak zorunda değildir.

4. Verilerin silinmesi

  • Uygulama içinden “Çıkış yap” veya benzeri işlemle bağlantıyı kestiğinizde, instance URL ve API anahtarı cihazdan silinebilir.
  • Uygulama verilerini tamamen kaldırmak için uygulamayı cihazınızdan kaldırmanız yeterlidir.

5. Güvenlik

API anahtarı ve instance URL’i, cihazın güvenli depolama alanında tutulur. İletişim, OpenProject sunucunuzla HTTPS üzerinden yapılır.

6. Değişiklikler

Bu politika güncellendiğinde “Son güncelleme” tarihi değiştirilir. Önemli değişiklikler varsa uygulama veya mağaza listesi üzerinden bilgi verilebilir.
