## OpenProject Mobile (Plan + Analysis)

Bu repo, OpenProject web deneyimini mobilde (online) sunmak için analiz + geliştirme çalışmalarını içerir.

### Dokümanlar

- `docs/analysis/`: İş isterleri (teknik çözüm değil)
- `docs/reviews/`: Kontrol / review çıktıları
- `docs/decisions/` ve `docs/adr/`: Karar kayıtları
- `docs/api/`: API keşfi ve sözleşmeler

### Çalışma prensibi

- Analiz dokümanlarında **nasıl yapılacağı** yazılmaz; sadece **ne istendiği** ve **kabul kriterleri** yazılır.
- Teknik tasarım/uygulama ayrıntısını işi alan agent belirler ve PR/commit açıklamalarında gerekçelendirir.
